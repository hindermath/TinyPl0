# Syntax-Referenz

Die offizielle EBNF fuer PL/0 befindet sich im kuratierten Kapitel aus `docs/LANGUAGE_EBNF.md`.
Hier wird eine kurze Uebersicht der wichtigsten Konstrukte gegeben.

- `const`: Konstantenblock
- `var`: Variablenblock
- `procedure`: Prozedurdefinition
- `begin ... end`: Block
- `if ... then`: Bedingung
- `while ... do`: Schleife
- `? ident`: Eingabe (extended)
- `! expression`: Ausgabe (extended)
