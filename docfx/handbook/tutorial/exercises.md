# Schritt-fuer-Schritt Uebungen

## Uebung 1: Addition

```text
LIT 0 2
LIT 0 3
OPR 0 2
```

Aufgabe: Fuege eine Speicherung in eine Variable hinzu.

## Uebung 2: Bedingter Sprung

```text
LIT 0 0
JPC 0 5
LIT 0 1
```

Aufgabe: Ergaenze den Code, sodass bei falscher Bedingung ein anderer Wert geladen wird.
