# Instruktions-Referenz

## Uebersicht

- LIT: Konstante laden
- LOD: Variable laden
- STO: Variable speichern
- CAL: Prozedur aufrufen
- INT: Stack reservieren
- JMP: Sprung
- JPC: Bedingter Sprung
- OPR: Operationen (Arithmetik, Vergleiche, Return)

Die vollstaendige Referenz wird aus dem VM-Instruction-Set kuratiert eingebunden.
