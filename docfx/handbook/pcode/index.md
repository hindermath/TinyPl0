# P-Code-Handbuch

Dieses Kapitel erklaert den P-Code der PL/0-VM und bietet Beispiele je Instruktion.
Die detaillierte Instruktionsliste wird im kuratierten Kapitel aus `docs/VM_INSTRUCTION_SET.md`
aufbereitet.

## Register und Speicher

- P: Program Counter
- B: Base Pointer
- T: Top-of-Stack
- I: Instruction Register

Die VM arbeitet mit einem Stack und einer separaten Code-Area.
