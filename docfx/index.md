# TinyPl0 Dokumentation

Willkommen zur Dokumentation von TinyPl0. Dieses Handbuch ist auf die Ausbildung ausgerichtet und kombiniert
Projekt- und Architekturwissen mit einem ausführlichen PL0- und P-Code-Handbuch sowie einer API-Referenz.

## Inhalte

- Einstieg, Installation und Nutzung der CLI.
- Architektur und Projektübersicht (Pl0.CLI, Pl0.Core, Pl0.Vm).
- PL0-Handbuch mit Beispielen auf drei Lernstufen.
- P-Code-Handbuch und Tutorial zur direkten P-Code-Programmierung.
- API-Referenz aus XML-Kommentaren.
- Anhang mit vollständigen Beispielprogrammen.

## Schnellstart

- Build: `dotnet build`
- Tests: `dotnet test`
- CLI-Run: `dotnet run --project src/Pl0.Cli -- run <datei.pl0>`

Weiter geht es mit dem Einstiegskapitel.
